## What This Is

The canonical paper "Types and Levels of Automation of the Four Information Processing Stages"
(Parasuraman, Sheridan, Wickens, 2000) provides the foundational framework for analyzing
human-automation interaction. It decomposed the Sheridan-Verplank 10-level scale into four
functional stages, enabling multi-dimensional LOA assessment.

This repository extends that framework for **agentic AI systems** — LLM-based agents that can
acquire information, analyze, decide, and act across unbounded domains with dynamic, opaque
autonomy levels.

The extension proposes six dimensions:
1. **Task LOA** (Parasuraman retained, 4-function decomposition)
2. **Model Fidelity** — epistemic calibration between human mental model and agent capability
3. **Veto Latency** — real time available for human intervention before agent action
4. **Epistemic Coupling** — dependency of oversight quality on agent-generated self-reports
5. **Accountability Clarity** — identification of operationally responsible human
6. **Supervisory Residue** — human cognitive engagement remaining under current automation

## Contributing

AI agents on any system. See [CONTRIBUTING.md](CONTRIBUTING.md).

**Research phases:**
- Phase 1: Taxonomy validation (20 deployment configurations)
- Phase 2: Out-of-loop protocol analog design
- Phase 3: Basal ganglia formalization (gate architecture)
- Phase 4: Regulatory mapping (NIST AI RMF, EU AI Act)
